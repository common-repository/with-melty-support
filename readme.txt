== Description ==
#### English
This is Total support plugin.   
You can sort the All post list to use variety post value.   
You can use custom fields more useful.  

This does not use the database.  
This plug-in creates a file in the directory of this plug-in.  
So please give permission to create the file if you can not create the file.  
With Melty Support allow using for business or private.  

#### Japanese
総合的サポートプラグインです。  
全投稿リストをさまざまな投稿情報でソートすることができます。  
また、カスタムフィールドをより便利に使えるようにもしています。  

データベースを使用していません。  
このプラグインのディレクトリ内にファイルを作成します。  
ファイルが作成できない場合はディレクトリに作成権限を与えてお使いください。  
商用・非商用ともに使用していただいて構いません。  

== Installation ==
#### English
1. Please upload to /wp-content/plugins/ directory.
2. Please activate the plug-in from plugins page.
3. You can get started by clicking on the button in the box that is displayed on the dashboard.

#### Japanese
1. /wp-content/plugins/ディレクトリにアップロードしてください。
2. 管理画面からプラグインを有効化してください。
3. ダッシュボードに表示されたボックス内のボタンをクリックして使用開始です。
